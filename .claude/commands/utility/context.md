# Load Project Context

Quickly load all context files for the Icarus project to ramp up on the current programme state.

## Steps

Read these files in parallel:

1. `context/glossary.md` — names, acronyms, systems, jargon
2. `context/stakeholders.md` — people, roles, dynamics
3. All files in `context/projects/` — per-project context
4. `context/decisions.md` — decision log
5. `context/open_threads.md` — unresolved questions
6. The most recent file in `analysis/weekly/` — latest weekly summary

## Summary

After reading, provide a brief status report:
- **Programme pulse**: overall momentum (accelerating/steady/decelerating)
- **Hot topics**: top 3 things being discussed this week
- **Key risks**: top 3 active risks
- **Upcoming**: what's expected next week
- **Attention needed**: anything that looks like it needs immediate action

Keep it to 10-15 lines max. This is a quick ramp-up, not a deep dive.
