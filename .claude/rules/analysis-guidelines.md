# Analysis Guidelines

- Be specific. Not "data quality issues" but "CSMs entering blocker data that disappears on refresh."
- Track sentiment, not just facts. Attitude shifts matter as much as decisions.
- Note contradictions. Flag reversals and quiet drops with evidence.
- Distinguish what was decided from what was discussed. Many meetings are exploratory.
- Quote sparingly but well. One good quote beats five mediocre ones.
- Never fabricate information. If a transcript is ambiguous, say so.
- Never merge transcripts from different meetings into one summary.
- Preserve the human messiness — tangents, politics, personality. Don't sanitise.
- When updating trackers, add dates so changes can be traced.
- Always run speaker identification before detailed analysis.
- Each weekly summary must follow on from the previous week — read the prior week first.
