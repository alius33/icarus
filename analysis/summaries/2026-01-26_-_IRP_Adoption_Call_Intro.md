# IRP Adoption Call — Portfolio Review Launch
**Date:** 2026-01-26
**Attendees:** Natalia (Plant), Ben Brooks, Diya Sawhny, Azmain Hossain, Rhonda, Josh Ellingson, multiple CSMs (broader team)
**Duration context:** Long (~1 hour)
**Workstreams touched:** WS2 CLARA, WS1 Training

## Key Points
- Landmark session: the first time the broader CSM team sees and interacts with CLARA
- Natalia presented the running agenda structure for Portfolio Review meetings — why the meeting exists, how it will run, who participates, call to action for feedback
- Ben Brooks gave the strategic overview: 12 months of building blocks (onboarding, training, blueprints, solution fit, teams) now mature enough to weave into a structured collaboration process
- Diya Sawhny spoke to position this as critical to the insurance scorecard and Andy Frappe's visibility requirements
- Natalia opened CSM name nominations for the tool — asking CSMs to register and update their account data
- Azmain demoed the app with Ben narrating — walked through dashboards, customer views, use cases, blocker tracking
- CSMs shown how to navigate the tool, update account data, manage blockers and action plans
- Ben pushed hard for CSMs to start entering data immediately rather than waiting for perfect data

## Decisions Made
- Portfolio Review will run weekly using CLARA as the primary tool → Natalia
- CSMs asked to nominate themselves in the tool and begin entering data for their accounts → All CSMs
- Data entry approach: start with existing Salesforce extract, CSMs augment and correct in real-time rather than waiting for clean data → Ben Brooks
