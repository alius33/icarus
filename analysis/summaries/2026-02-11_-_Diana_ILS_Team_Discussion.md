# Diana ILS Team Discussion — Project Management in CLARA
**Date:** 2026-02-11
**Attendees:** Azmain Hossain, Diana
**Duration context:** Medium (~27 minutes)
**Workstreams touched:** WS2 CLARA

## Key Points
- Diana is supporting the ILS (cat bond) team to adopt basic Microsoft tools. The team had not been using MS Teams at all and were managing documents in a completely manual, old-fashioned way — copying documents from one to another, no standardised processes.
- Diana is building Microsoft Lists and Power BI reporting for the ILS team. Some team members are enthusiastic early adopters; others find even basic channel usage awkward.
- Azmain floats the idea of using CLARA to replace the weekly advisory project review PowerPoint that Diana currently runs. Diana says the current Wednesday advisory meeting is "absolutely useless" — duplicate of a Monday meeting hosted by Arno/Arnold that covers the same project status and resources.
- Azmain's proposal: recreate the PowerPoint project status template as a screen within CLARA at the customer level. Project managers would go into a customer page, click "new" under a project management tab, get the previous week's updates pre-populated, make changes, and save a snapshot. Over time this builds a searchable history of project status week-by-week.
- Diana is enthusiastic about this approach. It would let CSMs see project status alongside use case data, giving visibility into the project management dimension that CLARA currently lacks.
- Cat bond projects are a separate category: different product (ILS not IRP), different challenges, high revenue. Diana suggests keeping cat bond tracking separate from CLARA. Azmain agrees to defer the cat bond piece.
- Azmain reveals he has requested to move into Diana's team when the org restructuring happens (rather than remain under Natalia). Diana is welcoming.
- Diana raises that only about 10% of the 30 priority clients have assigned project managers. The rest have no PM coverage — a bandwidth and strategic gap that needs escalation.
- Azmain uses the example of MS Ambian: it shows 90% green in CLARA, but anyone working on the project knows it is a nightmare. This demonstrates the missing project management visibility layer.
- Azmain commits to building the partner tracking feature for Alexandra/Liz McLagan this week (tomorrow and Friday), then starting v1 of the project management module next week.

## Decisions Made
- Project management status will be added to CLARA at the customer level, recreating the advisory review template -> Azmain to build v1
- Cat bond projects to be kept separate from CLARA (different product, different process) -> Diana/Azmain
- Advisory Wednesday meeting should be restructured or eliminated once project data moves into CLARA -> Diana to discuss with Stacy/Ben
